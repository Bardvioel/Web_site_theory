2. Flexbox - подробный справочник

Основы Flexbox

/* Grid настройки */
    display: flex;
    grid-template-columns: repeat(3, 1fr);
    gap: 16px;
    
    row-gap управляет расстоянием между строками,
    column-gap управляет расстоянием между столбцами.
    
    gap: 25px 15px; [промежуток высота длины]
    
    justify-content: center; /* Центрирование по горизонтали */
    align-items: center; /* Центрирование по вертикали */

Направление элементов (flex-direction)


    flex-direction: row — обычный ряд, элементы слева направо (по умолчанию)
    flex-direction: row-reverse — ряд в обратном порядке, справа налево
    flex-direction: column — колонка в обычном порядке, сверху вниз
    flex-direction: column-reverse — колонка в обратном порядке, снизу вверх

Выравнивание по поперечной оси (align-items)


    stretch — элементы растягиваются вдоль поперечной оси (по умолчанию)
    start — элементы прижимаются к началу поперечной оси
    end — элементы прижимаются к концу поперечной оси
    center — элементы выстраиваются по центру поперечной оси
    baseline — элементы выравниваются по базовой линии текста

Выравнивание по основной оси (justify-content)


    start — элементы в начале, пустое пространство после
    end — пустое пространство перед, элементы в конце
    center — элементы по центру, пространство поровну с двух сторон
    space-between — первый элемент в начале, последний в конце, остальные равномерно
    space-around — равные отступы вокруг каждого элемента
    space-evenly — все отступы одинаковой величины

Многострочное выравнивание (align-content)


    stretch — строки растягиваются равномерно (по умолчанию)
    start — все строки в начале родителя
    end — все строки в конце родителя
    center — строки по центру родителя
    space-between — первая строка в начале, последняя в конце
    space-around — равные отступы вокруг каждой строки
    space-evenly — одинаковые отступы между строками и краями

Перенос текста


    white-space: nowrap; /* текст не переносится */
    overflow-wrap: break-word; /* перенос длинных слов */

Порядок элементов (order)


    order: 0; /* значение по умолчанию */
    order: 1; /* элемент переместится в конец */
    order: -1; /* элемент будет первым в контейнере */

Гибкое изменение размеров


    flex-grow: 1; /* элемент забирает свободное пространство */
    flex-grow: 2; /* заберёт в 2 раза больше пространства чем flex-grow: 1 */
    
    flex-basis: auto; /* размер до расширения (по умолчанию) */
    flex-basis: 200px; /* базовый размер элемента */
    
    flex-shrink: 1; /* элемент сжимается (по умолчанию) */
    flex-shrink: 0; /* запрет на сжатие элемента */
            
    /* Сокращённая запись */
    flex: 0 1 auto; /* flex-grow: 0; flex-shrink: 1; flex-basis: auto; */

3. Позиционирование элементов

Относительное позиционирование


    /* Элемент остаётся в потоке, но смещается относительно самого себя */
    position: relative;
    top: 20px;
    right: 40px;
    bottom: 10px;
    left: 10px;

Абсолютное позиционирование


    /* Элемент вырывается из потока и позиционируется относительно ближайшего позиционированного предка */
    position: absolute;

Z-index - управление слоями


    /* Для управления близостью элементов используется свойство z-index */
    .background { z-index: -1; }
    .first-block { z-index: 0; }

Фиксированное позиционирование


    /* Элемент фиксируется относительно окна браузера */
    position: fixed;
    right: 0;
    left: 0;

Липкое позиционирование (sticky)


    /* Блок фиксируется при достижении края области просмотра */
    position: sticky;
    top: 20px;

Свойство inset


    /* Сокращённая запись для top, right, bottom, left */
    /* 10 пикселей сверху, 25 справа, 15 снизу и 0 слева */
    inset: 10px 25px 15px 0;

4. CSS Grid Layout

Основы Grid


    .container { 
        display: grid; 
    }

    /* Деление пространства на столбцы и строки */
    .container {
        display: grid;
        grid-template-columns: 33% 200px 270px;
        grid-template-rows: auto;
    }

    /* Использование fr единиц */
    .container {
        grid-template-columns: 1fr 1fr 1fr;
        grid-template-rows: auto;
    }

Отступы в Grid


    /* Создание отступов */
    gap: 10px;

    column-gap: 10px;
    row-gap: 20px;
    gap: 10px 20px; /* row-gap column-gap */

Функция repeat()


    grid-template-columns: repeat(/* количество столбцов, нужная ширина */);
    grid-template-rows: repeat(/* количество строк, нужная высота */);

    /* Примеры */
    grid-template-columns: repeat(5, 20%);
    grid-template-columns: repeat(5, 1fr);
    grid-template-columns: repeat(5, auto) 200px;

    .container {
        width: 600px;
        display: grid;
        grid-template-columns: 100px 2fr 3fr;
    }

Размещение элементов по линиям


    /* Свойства для размещения элементов */
    grid-column-start и grid-column-end — для вертикальных грид-линий;
    grid-row-start и grid-row-end — для горизонтальных линий.

    /* Пример */
    .container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        grid-template-rows: repeat(3, 1fr);
        gap: 40px 5px;
    }

    .block_size_big {
        grid-column-start: 1;
        grid-column-end: 3;
    }

    .block_size_big {
        grid-column-start: 1;
        grid-column-end: 3;
        grid-row-start: 1;
        grid-row-end: 6;
    }

    /* Сокращённая запись */
    .block_size_big {
        grid-column: 2/4;
    }

Именованные линии


    /* Именование линий */
    grid-template-rows: [aside-start] 300px [aside-end];

    /* Использование именованных линий */
    grid-row: aside-start / 4;

Грид-области


    /* Определение области */
    .header {
        grid-area: header;
    }

    /* Пример использования грид-областей */
    .container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        grid-template-rows: repeat(4, 1fr);
        grid-template-areas: 
            "header header header"
            "news news aside"
            "promo promo aside"
            ". footer footer";
    }

    .header {
        grid-area: header;
    }

Выравнивание контейнера


    /* Выравнивание содержимого грид-контейнера */
    .container {
        align-content: center;
        justify-content: space-between;
    }

    /* Сокращённая запись */
    .container {
        place-content: center space-between;
    }

    /* Всё по центру */
    .container {
        place-content: center;
    }

Выравнивание элементов


    /* Выравнивание элементов внутри ячеек */
    .container {
        justify-items: center; /* по горизонтали */
        align-items: center;   /* по вертикали */
    }

    /* Выравнивание конкретного элемента */
    .item {
        justify-self: start;   /* по горизонтали */
        align-self: end;       /* по вертикали */
    }

    /* Значения: stretch, start, end, center */

6. Командная строка (Terminal)

Основные команды навигации


    pwd — покажи, в какой я папке
    ls — покажи файлы в папке, где я сейчас
    cd first-project — перейди в папку first-project
    cd first-project/html — перейди в папку html, находящуюся в папке first-project
    cd .. — перейди на уровень выше в родительскую папку
    cd ~ — перейди в домашнюю директорию (у нас это /Users/stas_basov)

Работа с файлами и папками


    mkdir second-project — в текущей папке создай папку с именем second-project
    touch index.html — создай файл index.html в текущей папке
    touch index.html style.css script.js — создать несколько файлов через пробел
    rm about.html — удали файл about.html
    rmdir images — удали папку images
    rm -r second-project — удали папку second-project и всё, что она содержит

7. Git - система контроля версий

Синхронизация с удалённым репозиторием


    /* Привязка локального репозитория к удалённому */
    git remote add origin https://github.com/YandexPracticum/first-project.git

    /* Первая загрузка файлов в удалённый репозиторий */
    git push -u origin main

Создание коммитов (сохранений)


    /* Подготовка файлов к коммиту */
    git add название_файла — подготовить выбранный файл
    git add -A — подготовить все изменённые файлы

    /* Создание коммита с комментарием */
    git commit -m "комментарий к коммиту"

Публикация изменений


    /* Получение изменений от других разработчиков */
    git pull

    /* Отправка изменений в удалённый репозиторий */
    git push
