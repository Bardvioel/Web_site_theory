Все о веб-разработке

Основные технологии веб-разработки

  • HTML (структура)
  • CSS (стилизация)
  • JS (интерактивность)

Справочник по CSS

1. Базовые настройки

* { 
  box-sizing: border-box; 
}

html { 
  height: 100vh; 
}

2. Отступы и границы

Отступы:

/* Удаляет внутренние и внешние отступы */
margin: 0;      /* внешний отступ */
padding: 0;     /* внутренний отступ */

/* Для каждой стороны - своё значение */
padding: 20px 15px 30px 15px;

Границы:

/* Граница (цвет толщина начертание) */
border: 1px solid rgb(250 50 50 / 0.4);
/* Варианты начертания: dashed, double, dotted */

3. Размеры элементов

width: 500px;
height: 500px;

4. Работа с текстом

/* Шрифты */
font-family: 'Open Sans', sans-serif;
font-size: /* размер шрифта */;
font-weight: /* толщина шрифта */;
font-style: /* стиль шрифта, например italic */;
font-family: /* семейство шрифта */;

/* Трансформация текста */
text-decoration: underline;    /* подчёркнутый текст */
text-transform: uppercase;     /* ВЕРХНИЙ РЕГИСТР */
text-transform: lowercase;     /* нижний регистр */
text-transform: capitalize;    /* Каждое Слово С Заглавной */
text-align: center;            /* выравнивание */

5. Отображение и layout

/* Типы отображения */
display: block;        /* блочный элемент */
display: inline;       /* строчный элемент */
display: inline-block; /* блочно-строчный */
display: flex;         /* flexbox */
display: grid;         /* grid */

/* Grid настройки */
grid-template-columns: repeat(3, 1fr);
gap: 16px;

6. Визуальные эффекты

Тени:

box-shadow: -2px 2px 5px #FD6969;
/* сдвинута на 2px влево, на 2px вниз */
/* радиус размытия - 5px */
/* цвет красноватый, #FD6969 */

Фон:

/* Цвет фона */
background-color: forestgreen;
background-color: rgb(115 170 200 / 1);

/* Фото фон */
background-image: url(https://yastatic.net/s3/auth2/_/msoffice.d1ada09f.svg);
/* по центру */ background-position: center;
/* сожмёт или растянет сохранит пропорции */ background-size: cover;
/* сохранит пропорции */ background-size: contain;
/* первое значение - ширина, второе - высота */ background-size: 220px 400px;
/* так отменяется повтор изображения */ background-repeat: no-repeat;
/* изображение повторяется только по вертикали */ background-repeat: repeat-y;

7. Классы в HTML и CSS

<h1 class="text">123</h1> //html
.text {} //работа в CSS

8. Плавная смена текста (JavaScript)

//Плавно смена текста
smoothly(phrase, 'textContent', randomElement.text);
